# 🎬 Qt Video Player

一个基于 Qt 开发的视频播放器，支持播放列表、倍速播放、音量控制、全屏切换，并适配 PC 和嵌入式开发板环境。
使用前，请确保自己有相关的解码器，作者使用正点原子出厂镜像，已包含相关解码器，可以直接用。

---

## ✨ 功能特性

* 🎥 视频播放（支持 mp4 / avi / mkv）
* 📂 本地文件夹加载视频
* 📃 播放列表（QListWidget）
* ⏯ 播放 / 暂停 / 上一首 / 下一首
* ⏩ 倍速播放（0.5x / 1.0x / 1.5x / 2.0x）
* 🔊 音量控制（滑动条 + 外部同步）
* 🖥 全屏切换（因毕设需要，这个只能实现伪全屏效果，可以自己修改为真全屏）
* 🎯 播放进度控制（拖动进度条）
* 🧭 手势滑动列表（Gesture）
* ⏱ 引导页自动跳转

## 🧠 技术实现

* 使用 `QMediaPlayer` + `QMediaPlaylist` 实现播放控制
* 使用 `QVideoWidget` 进行视频渲染
* 使用 `QStackedWidget` 实现页面切换
* 使用 `QTimer` 控制 UI 行为（引导页 / 控制栏隐藏）
* 使用 `QButtonGroup` 实现倍速互斥选择
* 自定义 `Gesture` 实现上下滑动播放列表
* 自定义 `VoiceControl` 管理音量逻辑

---

## ⚠️ 已知问题

* 💻 PC端在窗口滑动/动画时可能出现卡顿（与硬件加速及视频渲染机制有关）
* 📺 不同平台下视频渲染表现不一致（开发板更稳定），开发板倍速功能不稳定，最好不要用
